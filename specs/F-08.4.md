## F-08.4 Street Name Lookup `[TODO]`

`app/geo.ts` — the only geolocation utility module. Creates the file from scratch. Exports one function: `getStreetName`.

`getStreetName(lat: number, lng: number): Promise<string | null>` — calls the Nominatim reverse-geocoding API with `zoom=17` (road-level resolution) and returns `data.address.road`. Returns `null` on network error or if the response does not include a road field (e.g., the user tapped a park or waterway). Uses `AbortController` with an 8-second timeout. Nominatim rate-limit: one request per second max; callers must not invoke this on every mouse move.

The `zoom=17` parameter is critical — without it Nominatim returns a city-level result rather than a specific road name.

**Tests** (in `tests/unit/geo.test.ts`)
- GIVEN Nominatim returns `{ address: { road: "Washington Street" } }`, WHEN `getStreetName` is called, THEN it returns `"Washington Street"`.
- GIVEN Nominatim returns `{ address: {} }` (no road field — e.g., a park), WHEN `getStreetName` is called, THEN it returns `null` without throwing.
- GIVEN Nominatim returns a network error, WHEN `getStreetName` is called, THEN it returns `null` without throwing.
- GIVEN the request takes longer than 8 seconds, WHEN the abort fires, THEN the function returns `null` and does not leave a dangling promise.
